# Soter

Soter是开源的PHP敏捷快速开发框架，支持“包”共享，HMVC，MVC的升级版DBVMC模式帮助你规范快速的开发你的项目，好学易用。  

## 手册

https://snail007.github.io/soter/  

## 快速入门

下载： https://github.com/snail007/soter/releases
  
然后解压到你的本地网站目录,比如/home/wwwroot/soter   

然后访问http://127.0.0.1/soter
