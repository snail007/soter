Soter代码规范
  1. 变量名，方法名，函数名，参数名，采取驼峰命名方式。比如userID
  2. 命名采用英文完整单词不能缩写，取最接近功能的单词。比如：userScoreLevel
  3. 常量名，静态变量名，采取全部大写加下划线分割多个单词的命名方式。比如：MAX_ID
  4. classes类目录下的类名和它的文件的位置是对应的。
    比如有文件：classes/Controller/User.php
    那么User.php文件里面的类名就应该是：Controller_User
    比如有文件：classes/Controller/Vip/User.php
    那么User.php文件里面的类名就应该是：Controller_Vip_User,也就是下划线代表着文件夹的分隔符。
    比如有文件：classes/Model/User.php
    那么User.php文件里面的类名就应该是：Model_User
    比如有文件：classes/MyLib/User.php
    那么User.php文件里面的类名就应该是：MyLib_User
